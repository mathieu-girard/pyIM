pyIM i a python module for displaying measurement from KEYENCE IM using the serial communication interface. It is multiplatform!

Installation:
- Install Python 3.x (tested with 3.3)
- Add library matplotlib (tested with 1.3.1)
- Add library numpy (tested with 1.8.1)
- Add library pyparsing (tested with 2.0.2)
- Add library dateutil (tested with 2.2)
- Add library six (tested with 1.7.2)
- Add library pyserial (tested with 2.7)

Usage:
- In a shell, python pyIM com_port baud_rate check_checksum e.g.: python com3 9600 1
